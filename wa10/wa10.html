Random Quotes

Extension Description

My extension to the basic quote machine includes using a different API (DummyJSON quotes API), adding a "show me who said it" feature where the quote is shown first and the user can click to see who said it, implementing a loading message while fetching data.

API Use in Critique Project

For my critique project on TikTok's algorithm, I could imagine using like a social media analytics API to pull data about video performance, trending hashtags, and/or engagement metrics. This would allow me to show how the algorithm prioritizes certain content type. By showing stuff like live data on metrics like view counts, shares, and watch time, I could create some kind of visualization maybe that helps users understand how TikTok's recommendation system works and what kind of stuff influences content visibility.
